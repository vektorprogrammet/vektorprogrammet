## Set up development environment
### Requirements
- [PHP](http://php.net/downloads.php) version >= 7.0
- [Node](https://nodejs.org/en/) version >= 4

#### UNIX:
`npm run setup`
#### Windows:
`npm run setup:win`

### Start server on http://localhost:8000
`npm start`
